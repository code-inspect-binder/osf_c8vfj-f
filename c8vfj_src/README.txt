Automatic Imitation Exp 1:
- Column 1: trialnr -- trial number
- Column 2: ppn -- participant number
- Column 3: age -- age
- Column 4: gender -- gender (m = male, v = female)
- Column 5: handpos -- position of the hands on the screen (0 = JB, 1 = PB, see stimuli)
- Column 6: group -- color of in-group (1 = in-group blue, 2 = in-group green)
- Column 7: block -- block number
- Column 8: cue -- cue (1 = W, 2 = P)
- Column 9: finger -- movement made by in-group hand (1 = index finger, 2 = little finger)
- Column 10: stim -- movements both hands (1 = left index, right little, 2 = left middle, right index)
- Column 11: color -- color of the left hand (1 = blue, 2 = green)
- Column 12: con -- congruency (0 = in-group incongruent, 1 = in-group congruent)
- Column 13: xr -- expected response (1 = index, 2 = little)
- Column 14: r -- actual response (1 = index, 2 = little)
- Column 15: corr -- accuracy (0 = incorrect, 1 = correct)
- Column 16: rt -- reaction time (in ms)
- Column 17: re -- error on reaction time (in ms)
